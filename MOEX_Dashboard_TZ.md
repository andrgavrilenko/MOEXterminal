## 1. ОБЩЕЕ ОПИСАНИЕ

### 1.1 Назначение

Интерактивный аналитический инструмент для поиска и анализа арбитражных
возможностей на Московской бирже (MOEX). Система отслеживает спотовые цены,
фьючерсы и скрытые ставки, выявляя математические несоответствия для получения
безрисковой (или низкорисковой) прибыли.

### 1.2 Область применения

Учебная разработка для понимания механик:
- Cash-and-Carry арбитража (спот + шорт фьючерс)
- Reverse C&C (шорт спот + лонг фьючерс)
- Валютных треугольных арбитражей
- Процентного паритета (CIP — Covered Interest Rate Parity)
- Carry вознаграждений на активах
- Календарных спредов (Calendar Spreads)
- Кривых доходности (Yield Curves)

### 1.3 Целевая аудитория

- Студенты финансовых факультетов
- Трейдеры и квант-аналитики
- Разработчики финтех приложений
- Исследователи рынков

## 3. ИСТОЧНИКИ ДАННЫХ (MOEX ISS API)

### 3.1 Базовые URL-ы

Все данные получаются бесплатно через REST API без авторизации.
Базовый URL: `https://iss.moex.com/iss/`

### 3.2 Необходимые эндпоинты

#### Спотовые курсы валют
```
GET /engines/currency/markets/selt/boards/CETS/securities.json
    ?iss.only=marketdata
    &securities=CNYRUB_TOM,USD000UTSTOM,EUR_RUB__TOM
```
Поля: `LAST` или `CURRENTVALUE` — последняя цена.

#### Кросс-курсы ЦБ
```
GET /statistics/engines/currency/markets/selt/rates.json
```
Поля: курсы ЦБ для USDCNY, EURUSD (расчётные).

#### Индексы (IMOEX, RTSI)
```
GET /engines/stock/markets/index/securities.json
    ?securities=IMOEX,RTSI
```
Поля: `CURRENTVALUE` — текущее значение индекса.

#### Фьючерсы (срочный рынок — FORTS)
```
GET /engines/futures/markets/forts/securities.json
    ?iss.only=securities,marketdata
```
Поля из `securities`: `SECID`, `SHORTNAME`, `LASTDELDATE` (дата экспирации),
`ASSETCODE`, `LOTSIZE`, `MINSTEP`, `STEPPRICE`.
Поля из `marketdata`: `LAST`, `BID`, `OFFER`, `SETTLEPRICE`.

#### Фьючерсные серии (для маппинга)
```
GET /statistics/engines/futures/markets/forts/series.json
```

#### Ставка RUSFAR
```
GET /statistics/engines/currency/markets/selt/rates.json
```
Или через индекс:
```
GET /engines/stock/markets/index/securities.json?securities=RUSFAR
```

#### Акции (спот)
```
GET /engines/stock/markets/shares/boards/TQBR/securities.json
    ?iss.only=marketdata
    &securities=HEAD,POSI,WUSH,X5,ALRS,AFKS,AFLT,BSPB,BANE,VKCO,...
```

#### Ключевая ставка ЦБ
Не через API — вводится вручную или парсится с сайта ЦБ РФ:
`https://www.cbr.ru/hd_base/KeyRate/`
На момент анализа: **КС ЦБ = 15.5%**

## 4. КЛЮЧЕВЫЕ ПАРАМЕТРЫ (Key Rate переменные)

Все расчеты в приложении завязаны на две критические переменные:

### 4.1 КС ЦБ (Ключевая ставка Центробанка)

| Параметр | Значение |
|---|---|
| Значение в примере | 15.5% |
| Обновление | От Центрального Банка России |
| Использование | Дисконтирование, Fair Value расчеты |
| Экономический смысл | Стоимость заимствования рублей |

Применение в формулах:
- `Fair Value = Spot * (1 + КС ЦБ * Days / 365)`
- `Carry Premium = Implied Rate - КС ЦБ`
- Baseline для сравнения всех implied rates

### 4.2 RUSFAR (Russian Secured Funding Average Rate)

| Параметр | Значение |
|---|---|
| Значение в примере | 15.15% |
| Обновление | От рынка (FX forward рынок) |
| Использование | Альтернативный baseline |
| Экономический смысл | Реальная ставка заимствования на рынке |

Разница КС ЦБ vs RUSFAR:
- КС ЦБ = 15.5% (инструмент ЦБ, более стабилен)
- RUSFAR = 15.15% (рыночная ставка)
- Разница = 0.35% (ликвидности и кредитного риска)

Применение в формулах:
- `Carry Premium = Implied Rate - RUSFAR` (альтернативный расчет)
- Для сравнения более "рыночной" доходности

## 5. АРБИТРАЖНЫЕ СТРАТЕГИИ

### 5.1 Обзор вкладки "Арбитраж"

> **Принцип:** Сходимость обеспечена экспирацией, треугольным соотношением
> или единым базовым активом. Прибыль = фиксированная разница на момент входа.

| Под-вкладка | Описание |
|---|---|
| Cash-and-Carry | Купить спот + продать фьючерс |
| Reverse C&C | Продать спот + купить фьючерс |
| Calendar FX | Спред между экспирациями одного валютного фьючерса |
| Cross-Instrument | Арбитраж между связанными фьючерсами |
| Cross-FX | Треугольный валютный арбитраж |
| Cross-Gold | Арбитраж золото RUB vs золото USD |
| CIP Arb | Covered Interest Rate Parity |
| Funding Analysis (Perpetual FX) | Фандинг вечных валютных фьючерсов vs cash rate |

---

### 5.2 Cash-and-Carry (C&C)

**Суть:** Купить базовый актив на споте, одновременно продать фьючерс.
При экспирации фьючерс сходится к споту — прибыль = разница цен минус затраты
на финансирование.

#### Механика стратегии (пошагово)

**День 0 (Начало):**
- КУПИТЬ спотовый актив (например, CNYRUB на спот 11.0935)
- ПРОДАТЬ фьючерс (контракт CRH6 на 11.1560)
- Зафиксировать спред

**День экспирации (день 31):**
- Получить актив по фьючерсу (как обещано)
- Прибыль = Фьюч Price - Spot Price - Carry Costs
- Carry Costs = Процентная ставка * Дни / 365

#### Формула Implied Rate

```
Implied_Rate = (Futures_Price / Spot_Price - 1) * (365 / Days_to_Expiry)
```

#### Пример расчёта (CNYRUB CRH6, пошагово)

```
Спот цена (CNYRUB):     11.0935
Цена фьючерса (CRH6):   11.1560
Дней до экспирации:      30

Шаг 1: Рассчитать изменение цены
  Изменение = 11.1560 / 11.0935 = 1.005617
  Процент изменения = 1.005617 - 1 = 0.005617 = 0.5617%

Шаг 2: Аннуализировать (привести к годовому показателю)
  Дневная ставка = 365 / 30 = 12.167
  Implied Rate = 0.005617 * 12.167 = 0.06617 = 6.617% годовых
```

**Интерпретация результата:**
- Implied Rate показывает, какую процентную ставку "закодирована" в фьючерсе
- Если Implied Rate < КС ЦБ — Фьючерс переоценен
  - Стратегия: ЛОНГ СПОТ + ШОРТ ФЬЮЧ (profiting from convergence)
- Если Implied Rate > КС ЦБ — Фьючерс недооценен
  - Стратегия: ШОРТ СПОТ + ЛОНГ ФЬЮЧ (обратная)

#### Fair Value (Справедливая стоимость фьючерса)

```
Fair Value = Spot * (1 + КС ЦБ * Days / 365)
```

Пример:
```
Fair Value = 11.0935 * (1 + 0.155 * 30 / 365)
           = 11.0935 * (1 + 0.01274)
           = 11.0935 * 1.01274
           = 11.2355

Текущий фьюч цена: 11.1560
Fair Value:         11.2355
Разница: Фьюч НИЖЕ fair value — переоценка спота или недооценка фьюча
```

#### Carry Premium (Вознаграждение за carry)

```
Carry Premium = Implied Rate - КС ЦБ
              = 6.617% - 15.5%
              = -8.883% (ОТРИЦАТЕЛЬНЫЙ)

Альтернативно vs RUSFAR:
Carry Premium = 6.617% - 15.15% = -8.533%
```

Интерпретация:
- Отрицательная премия = УБЫТОК от carry стратегии
- Положительная премия = ПРИБЫЛЬ от carry стратегии
- В примере: если купить спот по 11.0935 и продать фьюч по 11.1560,
  потеря ~8.88% на разнице между Implied (6.62%) и КС ЦБ (15.5%)

#### Таблица на экране (столбцы)

| Столбец | Описание | Формула |
|---|---|---|
| Актив | Базовый актив (CNYRUB, USDRUB, ...) | -- |
| Контракт | Тикер фьючерса (CRH6, SiM6, ...) | -- |
| Экспирация | Дата экспирации | из ISS API |
| Дней | Календарных дней до экспирации | `expiry_date - today` |
| Спот | Текущая цена базового актива | из API |
| Фьюч | Цена фьючерса | из API |
| Implied Rate | Подразумеваемая годовая ставка | `(Фьюч/Спот - 1) * 365/Дней` |

#### Арбитражные возможности

```
Условие: Implied_Rate > КС ЦБ (15.5%)
Действие: "Лонг спот + шорт фьюч"
Potential Profit = Implied_Rate - КС ЦБ
```

Пример из скриншота:
```
MIX Mix-3.26: implied 17.5% vs КС 15.5% → Лонг спот + шорт фьюч → +1.9%
MXI MXI-3.26: implied 17.5% vs КС 15.5% → Лонг спот + шорт фьюч → +1.9%
```

#### Цветовое кодирование Implied Rate

- Зелёный: implied rate > 0 (контанго, C&C прибылен)
- Красный: implied rate < 0 (бэквордация)
- Яркость пропорциональна абсолютному значению

### 5.3 Reverse Cash-and-Carry

**Суть:** Зеркальная стратегия — шорт спот + лонг фьючерс.
Выгодна когда implied rate < стоимости фондирования (бэквордация).

#### Механика стратегии (пошагово)

**День 0 (Начало):**
- ПРОДАТЬ спотовый актив в шорт (например, акцию Сбербанка)
- КУПИТЬ фьючерс на эту же акцию
- Зафиксировать спред

**День экспирации (когда фьючерс истекает):**
- Получить акцию по фьючерсу
- Вернуть заимствованную акцию
- Прибыль = Spot Price - Futures Price - Borrow Costs

#### Формула

```
Условие: Implied_Rate < RUSFAR (или Implied_Rate < 0)
Действие: "Шорт спот + лонг фьюч"
Potential Profit = RUSFAR - Implied_Rate
```

Carry Return:
```
Carry Return = [(Spot - Futures) / Spot] * (365 / Days) - Dividend Yield
```

#### Условия применения

1. Фьючерс НИЖЕ спота (бэквордация — BackWardation)
2. Акции с большими дивидендами
3. Временные рыночные аномалии

**Когда применяется:**
- Когда структура рынка инвертирована (BackWardation)
- Акции перед дивидендными выплатами (особенно Сбербанк, Газпром)
- Временные шоки, вызывающие неправильное ценообразование

---

### 5.4 Calendar FX (Календарный спред)

**Суть:** Арбитраж между двумя экспирациями одного фьючерса.
Спред между ближним и дальним контрактом должен соответствовать
разнице процентных ставок за период.

#### Механика

Одновременная торговля разными экспирациями одной валютной пары:
- Купить/продать контракт с близкой экспирацией
- Продать/купить контракт с дальней экспирацией
- Прибыль от сужения спреда между экспирациями

#### Паритет покрытых процентных ставок

```
Forward Premium = (Forward Rate / Spot Rate - 1) * (365 / Days)
Forward Premium = r_domestic - r_foreign
```

Для USDCNY:
```
Forward Premium должен быть ~ r_CNY - r_USD
```

Если Forward Premium постоянен между экспирациями — рынок справедлив.
Если Forward Premium различается — возможен арбитраж.

#### Формула Forward Rate (между двумя экспирациями)

```
Forward_Rate(T1->T2) = Futures(T2) / Futures(T1)
Implied_Forward_Rate = (Forward_Rate - 1) * 365 / (Days_T2 - Days_T1)
```

#### Арбитраж

```
Если Implied_Forward_Rate != ожидаемой ставке (RUSFAR):
  -> Купить ближний + продать дальний (или наоборот)
```

Пример:
```
H контракт (март):  Forward Premium = 5.0%
M контракт (июнь):  Forward Premium = 5.2%

Разница 0.2% -> Возможна торговля календарного спреда
```

### 5.5 Cross-Instrument

**Суть:** Арбитраж между фьючерсами на связанные активы.
Пример: фьючерс на IMOEX (MIX) vs мини-фьючерс на IMOEX (MXI)
— один и тот же индекс, разные контракты.

#### Механика

Один и тот же базовый актив, но торгуется разными контрактами.
Спред между ними должен соответствовать разнице в размерах контрактов.
Если спред ОТКЛОНЯЕТСЯ от нормы — арбитраж.

#### Примеры на MOEX

- MX vs MXI (оба IMOEX, но разные серии)
- RI vs RM (оба RTS Index, но разные размеры контракта)

#### Формула

```
Spread = MIX_price / MXI_price - 1
Если |Spread| > порог -> арбитраж
```

#### Пример из приложения

```
MIX Mix-3.26:  implied 17.9% vs КС 15.5% -> Potential Profit +2.43%
MXI MXI-3.26:  implied 17.7% vs КС 15.5% -> Potential Profit +2.24%

Действие: Лонг спот + шорт фьюч (обе серии)
```

### 5.6 Cross-FX (Треугольный валютный арбитраж)

**Суть:** Треугольник: USDRUB, CNYRUB и USDCNY.
Если кросс-курс, подразумеваемый из двух пар, отличается от спота
третьей пары — арбитраж.

#### Механика триангуляции

Три валютные пары образуют треугольник:
```
Si <-> CR <-> USDCNY <-> Si (замкнутый цикл)
```

#### Паритеты

**1. Основной паритет (Si Parity):**
```
Si (USDRUB) = CR (CNYRUB) * USDCNY

Пример:
  Si = 76.79 (по фактическим данным)
  CR = 11.0935
  USDCNY = 6.9218

  Справедливый Si = 11.0935 * 6.9218 = 76.78
  Разница = 76.79 - 76.78 = 0.01 (мало, рынок справедлив)
```

**2. Второй паритет (Eu Parity — добавляя евро):**
```
Eu (EURRUB) = Si (USDRUB) * EURUSD

Пример:
  Eu = 91.08
  Si = 76.79
  EURUSD = 1.1861

  Справедливое Eu = 76.79 * 1.1861 = 91.13
  Разница = 91.08 - 91.13 = -0.05 (небольшое отклонение)
```

#### Формулы

```
Implied_USDCNY = USDRUB / CNYRUB
Cross_Deviation = Implied_USDCNY / Spot_USDCNY - 1

Если Cross_Deviation > порог:
  Лонг CNYRUB + Шорт USDRUB + Шорт USDCNY
Если Cross_Deviation < -порог:
  Шорт CNYRUB + Лонг USDRUB + Лонг USDCNY
```

#### Когда возникает арбитраж

Если ФАКТИЧЕСКОЕ Si != CR * USDCNY:
- Можно купить дешевый инструмент
- Продать дорогой инструмент
- Получить прибыль от конвергенции

#### Пример торговли

Допустим Si переоценен (Si > CR * USDCNY):
1. Купить CR (дешево, базовый актив)
2. Купить USDCNY (пара, связанная с Si)
3. Продать Si (дорого)
4. Получить прибыль при конвергенции

### 5.7 Cross-Gold (Арбитраж золота)

**Суть:** Золото торгуется в двух валютах — GD (USD за унцию) и GL (RUB за грамм).
При известном курсе USDRUB и коэффициенте перевода (1 тр. унция = 31.1035 г)
можно найти расхождение.

#### Формула паритета

```
GL (Price in RUB) = GD (Price in USD) * USDRUB * Conversion Factor

где Conversion Factor = 1 troy oz / 1 gram = 31.1035
```

Или эквивалентно:
```
GL_fair = GD_price * USDRUB / 31.1035
Deviation = GL_market / GL_fair - 1

Если |Deviation| > порог -> арбитраж
```

#### Пример из данных

```
Gold (USD) базовый контракт (GDH6):     5023.50 USD/oz
Gold (RUB) базовый контракт (GLH6):    12478.30 RUB/g
USDRUB спот:                            76.7900

Implied USDRUB = GL / (GD * 31.1035)
               = 12478.30 / (5023.50 * 31.1035)
```

#### Когда возникает арбитраж

Если GL/GD != USDRUB * 31.1035:
- Возможна покупка дешевого золота (в одной валюте)
- Продажа дорогого (в другой валюте)
- Получение прибыли от конвергенции цен

### 5.8 CIP Arb (Covered Interest Rate Parity)

**Суть:** Паритет покрытых процентных ставок.
Implied forward USDCNY рассчитывается через фьючерсы Si (USDRUB) и CR (CNYRUB).
Расхождение forward premium между экспирациями = арбитраж.

#### Концепция

Паритет покрытых процентных ставок гарантирует, что прибыль от:
  "Заимствовать в USD + конвертировать в CNY + вложить в CNY"
равна:
  "Держать USD с процентом USD"

#### Математическая форма

```
(Forward Rate / Spot Rate - 1) * (365 / Days) = r_domestic - r_foreign

Для Si/CR (USDCNY):

Forward Premium CNY = (Implied USDCNY forward / Spot USDCNY - 1) * (365 / Days)
                    ~ r_CNY - r_USD
```

#### Как считается Implied Rate

Из Si и CR фьючерсов рассчитывается IMPLIED USDCNY forward:

```
# 1. Implied USDCNY через фьючерсы
Impl_USDCNY(T) = Si(T) / CR(T)

где:
  Si(T) = цена фьючерса на USDRUB с экспирацией T
  CR(T) = цена фьючерса на CNYRUB с экспирацией T

# 2. Forward Premium (годовой)
Fwd_Premium(T) = (Impl_USDCNY(T) / Spot_USDCNY - 1) * 365 / Days(T)

# По теории CIP, Fwd Premium должен быть стабилен и примерно равен:
Fwd_Premium ~ r_CNY - r_USD

# 3. Арбитраж: отклонение от среднего
Avg_Fwd_Premium = среднее по всем экспирациям
Deviation(T) = Fwd_Premium(T) - Avg_Fwd_Premium

Если Deviation(T) значительно != 0:
  -> арбитражная возможность
```

#### Таблица CIP из скриншота

| Экспирация | Дней | Si (USDRUB fwd) | CR (CNYRUB fwd) | Impl. USDCNY | Spot USDCNY | Fwd Premium |
|---|---|---|---|---|---|---|
| 03.2026 | 30 | 76.7380 | 11.1560 | 6.8786 | 6.9218 | -7.59% |
| 06.2026 | 121 | 78.6000 | 11.4380 | 6.8718 | 6.9218 | -2.18% |
| 09.2026 | 212 | 80.5940 | 11.7090 | 6.8831 | 6.9218 | -0.96% |
| 12.2026 | 303 | 83.1900 | 12.0270 | 6.9169 | 6.9218 | -0.08% |
| 03.2027 | 394 | 86.3150 | 12.4200 | 6.9497 | 6.9218 | +0.37% |
| 06.2027 | 485 | 89.2010 | 12.7270 | 7.0088 | 6.9218 | +0.95% |

#### Действия при арбитраже

| Fwd Premium | Действие | Прибыль (пример) |
|---|---|---|
| Сильно отрицательный (-7.6%) | Лонг Si + Шорт CR*7 + Лонг USDCNY спот | +6.01% |
| Положительный (+0.9%) | Шорт Si + Лонг CR*7 + Шорт USDCNY спот | +2.53% |

> При USDCNY ~ 7 -> 1 контракт Si ~ 7 контрактов CR по номиналу.

### 5.9 Funding Analysis (Perpetual FX)

**Суть:** сравнение дневного фандинга вечных валютных фьючерсов с implied rate по датированным фьючерсам и cash rate (КС ЦБ / RUSFAR).

**Инструменты:**
- USDRUBF, EURRUBF, CNYRUBF (вечные фьючерсы)
- Спот: USDRUB, EURRUB, CNYRUB
- Cash rate: КС ЦБ и/или RUSFAR

**Ключевые величины:**
```
D  = avg(PerpPrice - Spot) за минутные окна 10:00 → клиринг (без промежуточного клиринга)
L1 = K1 * Spot
L2 = K2 * Spot

Funding = MIN(L2; MAX(-L2; MIN(-L1, D) + MAX(L1, D)))
```

**Предикт (знак фандинга):**
```
if |D| < L1 -> Predicted Funding = 0
else        -> Predicted Funding = sign(D) (capped by L2)
```

**Рекомендация по позиции (perpetual):**
- Predicted Funding > 0  -> SHORT (шорты получают фандинг)
- Predicted Funding < 0  -> LONG (лонги получают фандинг)
- Predicted Funding = 0  -> NEUTRAL

**Сравнение с альтернативами:**
```
Implied_Rate = (Futures_Price / Spot_Price - 1) * 365 / Days_to_Expiry
Funding_Rate_Daily = Funding / Spot
Funding_Annualized = Funding_Rate_Daily * 365
Funding_vs_Cash = Funding_Annualized - Cash_Rate
Implied_vs_Cash = Implied_Rate - Cash_Rate
```

**Источники данных (MOEX ISS):**
```
Spot (USD/RUB TOM):
GET https://iss.moex.com/iss/engines/currency/markets/selt/securities/USD000UTSTOM.json

Spot (EUR/RUB TOM):
GET https://iss.moex.com/iss/engines/currency/markets/selt/securities/EUR_RUB__TOM.json

Spot (CNY/RUB TOM):
GET https://iss.moex.com/iss/engines/currency/markets/selt/securities/CNYRUB_TOM.json

Perpetual candles (пример: USDRUBF):
GET https://iss.moex.com/iss/engines/futures/markets/forts/securities/USDRUBF/candles.json?interval=1

Spot candles (пример: USD/RUB):
GET https://iss.moex.com/iss/engines/currency/markets/selt/securities/USD000UTSTOM/candles.json?interval=1
```
Indicative funding (swap_rate) доступен через шлюз FORTS_COMMON_REPL (FAST-протокол MOEX).

**Параметры контрактов (K1, K2, LOTSIZE):**
```
Perpetual specs (пример: USDRUBF):
GET https://iss.moex.com/iss/engines/futures/markets/forts/securities/USDRUBF.json?iss.only=securities
```
Ищем поля: `K1`, `K2`, `LOTSIZE` (могут называться иначе, если биржа меняет спецификацию).

**Fallback, если swap_rate недоступен:**
- Использовать расчетный Predicted Funding по `D/L1/L2` и помечать Indicative Funding как "N/A".
- Опционально: считать Indicative Funding как `Funding` по текущему `D` (минутные свечи perp/spot).

**Качество данных (minute candles):**
- Если часть минут отсутствует -> считать D по доступным минутам и помечать "partial".
- Если данных < 60 минут -> не показывать Predicted Funding (insufficient).
- Если spot/perp таймстемпы расходятся > 2 мин -> пропустить эту минуту.

**Форматирование и цветовые правила (Funding Analysis):**
- Funding / Predicted Funding:
  - > 0 -> зеленый фон (SHORT выгоден)
  - < 0 -> красный фон (LONG выгоден)
  - = 0 -> нейтральный серый
- Suggested Action:
  - SHORT -> зеленый
  - LONG -> красный
  - NEUTRAL -> серый
- Funding vs Cash / Implied vs Cash:
  - > 0 -> зеленый
  - < 0 -> красный
  - около 0 (|x| < 0.10%) -> серый/желтый
- Indicative Funding:
  - если "N/A" -> серый, курсив

**Таблица (столбцы):**
- Валюта, Spot, Perp, D, L1, L2
- Funding (daily), Indicative Funding (swap_rate)
- Predicted Funding (sign), Suggested Action
- Implied Rate (dated futures), Cash Rate
- Funding vs Cash, Implied vs Cash

## 6. ВКЛАДКА: RELATIVE VALUE

> **Принцип:** Нет гарантии конвергенции. Сезонность, хранение, carry premium.
> Это направленные ставки на изменение спреда.

### 6.1 Под-вкладки

| Под-вкладка | Описание |
|---|---|
| Calendar Commodity | Календарные спреды товаров |
| Curve Shape | Форма кривой (контанго/бэквордация) |
| Rate Spread | Спред ставок между активами |

### 6.2 Curve Shape — контанго / бэквордация

**Суть:** Анализ формы фьючерсной кривой для каждого актива.

**Формула:**
```
Спред = Back_Price - Front_Price

Где:
  Front = ближайший контракт (наименьшие дни до экспирации)
  Back  = самый дальний контракт

Форма кривой:
  Если Спред > 0  -> КОНТАНГО (фьючерс дороже спота)
  Если Спред < 0  -> БЭКВОРДАЦИЯ (фьючерс дешевле спота)
  Если кривая немонотонна -> СМЕШАННАЯ
```

**Таблица (столбцы):**

| Столбец | Описание |
|---|---|
| Актив | Название (CNYRUB, USDRUB, Brent, Gold...) |
| Контр. | Количество доступных фьючерсных контрактов |
| Front | Ближайший контракт: тикер=цена |
| Back | Дальний контракт: тикер=цена |
| Форма | КОНТАНГО / БЭКВОРДАЦИЯ / СМЕШАННАЯ |
| Спред | Back_Price - Front_Price (в единицах актива) |

**Группировка по классам:**
- Валюты: CNYRUB, USDRUB, EURRUB
- Индексы: MIX, MXI, RTS, RTSM
- Энергоносители: Brent, Nat Gas
- Металлы: Gold (USD), Gold (RUB), Silver, Platinum, Palladium

## 7. ВКЛАДКА: КРИВЫЕ

> Детальные фьючерсные кривые по каждому выбранному активу.

### 7.1 Принцип

Для каждого актива показывается полная таблица всех доступных фьючерсных контрактов
с расчётом implied rate и премии к КС ЦБ.

**Важное разделение:**
- **Валюты/индексы** — implied rate считается от **спота**
- **Товары** — implied rate считается от **фронтального (ближайшего) контракта**

### 7.2 Формулы для валют и индексов

```
Implied_Rate = (Futures_Price / Spot_Price - 1) * 365 / Days

Премия_к_КС = Implied_Rate - КС_ЦБ
```

**Пример: CNYRUB (спот: 11.0935)**

| Фьючерс | Экспирация | Дней | Цена | Implied Rate | Премия к КС |
|---|---|---|---|---|---|
| CRH6 | 2026-03-19 | 30 | 11.1560 | +6.85% | -8.65% |
| CRM6 | 2026-06-18 | 121 | 11.4380 | +9.37% | -6.13% |
| CRU6 | 2026-09-17 | 212 | 11.7090 | +9.55% | -5.95% |
| CRZ6 | 2026-12-17 | 303 | 12.0270 | +10.14% | -5.36% |
| CRH7 | 2027-03-18 | 394 | 12.4200 | +11.08% | -4.42% |
| CRM7 | 2027-06-17 | 485 | 12.7270 | +11.08% | -4.42% |

**Проверка расчёта CRH6:**
```
Implied Rate = (11.1560 / 11.0935 - 1) * 365 / 30
             = (1.005634 - 1) * 12.1667
             = 0.005634 * 12.1667
             = 0.06854 = 6.85% ✓

Премия к КС = 6.85% - 15.5% = -8.65% ✓
```

### 7.3 Формулы для товаров

```
# Для товаров нет спота на MOEX, поэтому base = фронтальный контракт
Implied_Rate = (Futures_Price / Front_Price - 1) * 365 / (Days - Days_Front)
```

### 7.4 Формулы для акций (расширенные)

Для акций добавляются колонки Fair value и дивиденды:

```
# Fair value = теоретическая цена фьючерса без дивидендов
Fair = Spot * (1 + КС * Days / 365)

# Implied дивиденд = Fair - Futures
Div_Implied = Fair - Futures_Price

# Дивидендная доходность
Div_Yield = Div_Implied / Spot

# Implied Rate (как для всех)
Implied_Rate = (Futures_Price / Spot - 1) * 365 / Days

# Премия к КС
Premium_vs_KS = Implied_Rate - КС_ЦБ
```

**Пример: HeadHunter (спот: 2917.00 руб.)**

| Фьючерс | Дней | Цена | Fair (без див) | Implied Rate | Премия к КС | Div руб. | Div % |
|---|---|---|---|---|---|---|---|
| HDH6 | 31 | 2951.00 | 2955.40 | +13.72% | -1.78% | 4.40 | 0.15% |
| HDM6 | 122 | 2885.00 | 3068.12 | -3.28% | -18.78% | 183.12 | 6.28% |

**Проверка расчёта HDH6:**
```
Fair = 2917.00 * (1 + 0.155 * 31/365) = 2917.00 * 1.01317 = 2955.40 ✓
Div_Implied = 2955.40 - 2951.00 = 4.40 ✓
Div% = 4.40 / 2917.00 = 0.15% ✓
Implied Rate = (2951/2917 - 1) * 365/31 = 0.01166 * 11.774 = 13.72% ✓
```

**Интерпретация для HDM6:**
```
Fair = 2917.00 * (1 + 0.155 * 122/365) = 2917.00 * 1.05181 = 3068.12
Div_Implied = 3068.12 - 2885.00 = 183.12 руб.
-> Рынок ожидает дивиденд ~183 руб. на акцию до июня 2026
-> Implied Rate отрицательный (-3.28%), потому что фьючерс ниже спота
   из-за ожидаемых дивидендов
```

### 7.5 Цветовое кодирование

| Метрика | Зелёный фон | Красный/розовый фон |
|---|---|---|
| Implied Rate | > 0% | < 0% |
| Премия к КС | > 0% (дороже КС) | < 0% (дешевле КС) |
| Fwd Premium | ~ 0 (вблизи паритета) | Далеко от 0 |

Интенсивность цвета пропорциональна абсолютному значению.

## 9. СВОДКА ВСЕХ ФОРМУЛ

### 9.1 Формула #1: IMPLIED RATE (Основная формула — Скрытая процентная ставка)

**Формула:**
```
Implied Rate = (Futures Price / Spot Price - 1) * (365 / Days to Expiration)
```

**Для товаров (от фронтального контракта):**
```
Implied Rate = (Futures Price / Front Price - 1) * 365 / (Days - Days_Front)
```

**Математическое значение:** Показывает, какую годовую процентную ставку "закодирована" в фьючерсе.

**Пример расчёта (CNYRUB CRH6, пошагово):**
```
Спот цена (CNYRUB):            11.0935
Цена фьючерса (CRH6):          11.1560
Дней до экспирации:            30

Шаг 1: Рассчитать изменение цены
  Изменение = 11.1560 / 11.0935 = 1.005617
  Процент изменения = 1.005617 - 1 = 0.005617 = 0.5617%

Шаг 2: Аннуализировать (привести к годовому показателю)
  Дневная ставка = 365 / 30 = 12.167
  Implied Rate = 0.005617 * 12.167 = 0.06617 = 6.617% годовых
```

**Интерпретация:** Если взять рубли в кредит на 30 дней при ставке 6.617% годовых,
получится ровно спотовая цена к цене фьючерса.

**Использование в стратегиях:**
- Cash-and-Carry: ЛОНГ спот (купить за кредит) + ШОРТ фьюч
- Reverse C&C: ШОРТ спот + ЛОНГ фьюч
- Сравнение: Implied vs КС ЦБ для поиска неправильного ценообразования

---

### 9.2 Формула #2: FAIR VALUE (Справедливая стоимость)

**Формула:**
```
Fair Value = Spot Price * (1 + Key Rate * Days to Expiration / 365)
```

**С дивидендами (для акций):**
```
Fair Value (adj) = Spot * (1 + Key Rate * Days / 365) - PV(dividends)
```

**Математическое значение:** Теоретическая цена фьючерса при полностью справедливом ценообразовании.

**Пример расчёта (CNYRUB):**
```
Спот цена (CNYRUB):          11.0935
КС ЦБ:                       15.5% = 0.155
Дней до экспирации:          30

Fair Value = 11.0935 * (1 + 0.155 * 30 / 365)
           = 11.0935 * (1 + 0.01274)
           = 11.0935 * 1.01274
           = 11.2355
```

**Интерпретация:** При справедливом ценообразовании фьючерс CRH6 должен стоить 11.2355.
Фактическая цена: 11.1560. Вывод: фьючерс НЕДООЦЕНЕН на 0.0795 (примерно 0.7%).

**Использование:**
- Определение переоценки/недооценки фьючерса
- Определение справедливой прибыли от carry стратегии

---

### 9.3 Формула #3: CARRY PREMIUM (Вознаграждение за carry)

**Формула:**
```
Carry Premium = Implied Rate - Key Rate
```

Или альтернативно:
```
Carry Premium = Implied Rate - RUSFAR
```

**Математическое значение:** Дополнительный доход (или убыток) от carry стратегии сверх "справедливой" ставки.

**Пример расчёта (CNYRUB CRH6):**
```
Implied Rate (рассчитано выше):  6.617%
КС ЦБ:                           15.5%

Carry Premium = 6.617% - 15.5% = -8.883% (ОТРИЦАТЕЛЬНЫЙ)

или с RUSFAR:
Carry Premium vs RUSFAR = 6.617% - 15.15% = -8.533%
```

**Интерпретация:**
- Отрицательная премия = УБЫТОК от carry стратегии
- Положительная премия = ПРИБЫЛЬ от carry стратегии
- В примере: если купить спот по 11.0935 и продать фьюч по 11.1560,
  потеря ~8.88% на разнице между Implied (6.62%) и КС ЦБ (15.5%)

**Визуальное отображение в приложении:**
- Зеленый цвет: Positive Carry Premium (можно торговать)
- Красный цвет: Negative Carry Premium (нет возможности арбитража)

---

### 9.4 Формула #4: BASIS (Базис)

**Формула:**
```
Basis = Futures Price - Spot Price
```

**Процентный базис:**
```
Basis % = (Basis / Spot Price) * 100%
```

**Пример расчёта (CNYRUB CRH6):**
```
Фьючерс цена:     11.1560
Спот цена:        11.0935

Basis = 11.1560 - 11.0935 = 0.0625
Basis % = (0.0625 / 11.0935) * 100% = 0.563%
```

**Интерпретация:**
- Фьючерс торгуется с ПРЕМИЕЙ (Basis > 0) — это Contango (нормальная кривая)
- Если Basis < 0 — это BackWardation (инвертированная кривая)

**Использование:**
- Определение структуры кривой (Contango vs BackWardation)
- Базис для расчета carry

---

### 9.5 Формула #5: DAILY CARRY (Ежедневное вознаграждение)

**Формула:**
```
Daily Carry = (Basis / Spot Price) / Days * 365
            = (Futures - Spot) / Spot / Days * 365
```

**Пример расчёта (CNYRUB CRH6):**
```
Basis:            0.0625
Спот цена:        11.0935
Дней:             30

Daily Carry = (0.0625 / 11.0935) / 30 * 365
            = 0.005617 / 30 * 365
            = 0.06617 = 6.617% (это IMPLIED RATE)
```

**Интерпретация:** Показывает, какой процент в день вы зарабатываете от carry стратегии
(если аннуализировать ежедневный carry).

---

### 9.6 Формула #6: FORWARD PREMIUM (Форвард премия)

**Формула:**
```
Forward Premium = (Forward Rate / Spot Rate - 1) * (365 / Days)
```

**Теория паритета процентных ставок (CIP):**
```
Forward Premium ~ r_domestic - r_foreign

Для USDCNY:
Forward Premium CNY = (Implied USDCNY forward / Spot USDCNY - 1) * (365 / Days)
                    ~ r_CNY - r_USD
```

**Пример расчёта:**
```
Spot USDCNY:              6.9218
Implied USDCNY (forward): 7.0000
Дней:                     91

Forward Premium = (7.0000 / 6.9218 - 1) * (365 / 91)
                = (1.01127 - 1) * 4.011
                = 0.01127 * 4.011
                = 0.0452 = 4.52% годовых
```

**Интерпретация:** Юань торгуется с ФОРВАРД ПРЕМИЕЙ в 4.52%.
Это означает, что ставка в Китае выше, чем в США (примерно на 4.52%).

**Использование:**
- Calendar Spread FX торговля
- CIP Arbitrage поиск
- Анализ валютного рынка

---

### 9.7 Формула #7: SI PARITY (Паритет Si = CR * USDCNY)

**Формула:**
```
Si (USDRUB) = CR (CNYRUB) * USDCNY
```

Или для фьючерсов на конкретную дату:
```
Si.futures = CR.futures * USDCNY.futures
```

**Пример проверки паритета:**
```
CR (CNYRUB спот):         11.0935
USDCNY (спот):            6.9218

Справедливое Si = 11.0935 * 6.9218 = 76.776

Si (фактический спот):    76.79

Разница = 76.79 - 76.776 = 0.014 (небольшое отклонение, нормально)
```

**Использование:**
- Cross-FX (треугольный) арбитраж
- Определение переоценки/недооценки Si
- Проверка справедливого ценообразования

---

### 9.8 Формула #8: EU PARITY (Паритет Eu = Si * EURUSD)

**Формула:**
```
Eu (EURRUB) = Si (USDRUB) * EURUSD
```

**Пример проверки паритета:**
```
Si (USDRUB спот):         76.79
EURUSD (спот):            1.1861

Справедливое Eu = 76.79 * 1.1861 = 91.125

Eu (фактический спот):    91.08

Разница = 91.08 - 91.125 = -0.045 (небольшое отклонение)
```

**Использование:**
- Cross-FX треугольный арбитраж (расширение)
- Определение переоценки/недооценки Eu
- Комплексный анализ валютных рынков

---

### 9.9 Сводка формул (кросс-курсы, спреды, прочее)

```python
# === КРОСС-КУРС ===
usdcny_implied = usdrub / cnyrub
eurusd_implied = eurrub / usdrub

# === CIP (Covered Interest Parity) ===
impl_usdcny_fwd = si_futures / cr_futures
fwd_premium = (impl_usdcny_fwd / spot_usdcny - 1) * 365 / days
# Должен быть ~ (r_CNY - r_USD)

# === СПРЕД КРИВОЙ ===
spread = back_price - front_price
# > 0 -> контанго, < 0 -> бэквордация

# === CASH-AND-CARRY PROFIT ===
cc_profit = implied_rate - funding_rate  # (RUSFAR или КС)
# Если > 0: лонг спот + шорт фьюч = прибыль

# === FORWARD RATE (между экспирациями) ===
fwd_rate = (futures_far / futures_near - 1) * 365 / (days_far - days_near)

# === IMPLIED DIVIDEND ===
div_implied = fair_value - futures_price
div_yield = div_implied / spot

# === PREMIUM К КЛЮЧЕВОЙ СТАВКЕ ===
premium_vs_ks = implied_rate - key_rate
```

## 11. РЕАЛИЗАЦИЯ НА PYTHON (псевдокод)

```python
import pandas as pd
from datetime import datetime, date

def calc_days_to_expiry(expiry_date: date) -> int:
    return (expiry_date - date.today()).days

def calc_implied_rate(futures: float, spot: float, days: int) -> float:
    if days <= 0 or spot <= 0:
        return None
    return (futures / spot - 1) * 365 / days

def calc_fair_value(spot: float, key_rate: float, days: int) -> float:
    return spot * (1 + key_rate * days / 365)

def calc_div_implied(fair: float, futures: float) -> float:
    return fair - futures

def calc_fwd_premium(impl_cross: float, spot_cross: float, days: int) -> float:
    if days <= 0 or spot_cross <= 0:
        return None
    return (impl_cross / spot_cross - 1) * 365 / days

def determine_curve_shape(prices: list[float]) -> str:
    if len(prices) < 2:
        return "N/A"
    diffs = [prices[i+1] - prices[i] for i in range(len(prices)-1)]
    if all(d > 0 for d in diffs):
        return "КОНТАНГО"
    elif all(d < 0 for d in diffs):
        return "БЭКВОРДАЦИЯ"
    else:
        return "СМЕШАННАЯ"

GET https://iss.moex.com/iss/statistics/engines/stock/markets/index/analytics/{CODE}.json?iss.only=analytics&analytics.columns=ticker,shortnames,weight&limit=100
```
Где `{CODE}` = MOEXEU, MOEXOG, MOEXMM, MOEXFN, MOEXCN, MOEXTL, MOEXTN, MOEXCH.

**Историческая котировка (для расчёта моментумов):**
```
GET https://iss.moex.com/iss/history/engines/stock/markets/index/securities/{CODE}.json?from=2025-02-23&till=2026-02-23&iss.only=history&history.columns=TRADEDATE,CLOSE
```
