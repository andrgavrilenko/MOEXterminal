> Документ создан 26.02.2026 на основе автоматического обхода https://www.packmanmarkets.ru/

## 1. Что такое Packman Monitor (простым языком)

**Packman Monitor** — это бесплатный аналитический терминал для Московской биржи. Работает в браузере, обновляется каждые 3 секунды, показывает всё: от курса доллара до арбитражных возможностей на фьючерсах.

**Для кого:** трейдеры срочного рынка, портфельные инвесторы, те кто торгует облигации.

**Чем отличается от TradingView / Smart-Lab:** фокус на *арбитраже* — портал сам находит расхождения цен между спотом и фьючерсами, валютные треугольники, аномалии в OI. Это как Bloomberg Terminal для MOEX, только бесплатный.

## 2. Разделы портала (простым языком)

### 2.1 Акции

| Подраздел | Что делает | Кому полезно |
|---|---|---|
| **Market Data** | Таблица всех ~250 акций MOEX: цена, оборот, динамика, дивиденды, дисбаланс покупок/продаж | Всем — базовый скрининг |
| **Dividend Arbitrage** | Для каждой акции с фьючерсом считает implied rate, carry, ожидаемый дивиденд. Показывает, выгоднее ли купить акцию или фьючерс | Трейдерам вокруг дивидендных отсечек |
| **Fundamentals** | Полный фундаментальный анализ: P/E, ROE, EBITDA, баланс, денежные потоки. 414 компаний, МСФО и РСБУ, годовые и квартальные | Долгосрочным инвесторам |
| **Consensus** | Консенсус-прогнозы аналитиков: целевые цены, рекомендации, мультипликаторы. 124 акции | Инвесторам для отбора идей |
| **Volume** | Аномалии объёмов торгов: где объём сильно выше среднего | Трейдерам — ранний сигнал движения |

### 2.2 Облигации

| Подраздел | Что делает | Кому полезно |
|---|---|---|
| **ОФЗ** | 30 выпусков ОФЗ + интерактивная кривая доходности (текущая vs история за 13 лет). Средний YTM ~14.4% при КС 15.5% — рынок ждёт снижения ставки | Облигационным инвесторам |
| **Корп. облигации** | 500+ бумаг с фильтрами по рейтингу, сектору, эмитенту. Scatter-plot доходность vs дюрация. G-spread над ОФЗ | Для построения портфеля облигаций |

### 2.3 Фьючерсы

| Подраздел | Что делает | Кому полезно |
|---|---|---|
| **Market Data** | 270+ контрактов FORTS: цена, VWAP, OI, объём, дисбаланс, ГО. Фильтр по категориям | Трейдерам деривативов |
| **Curves** | Графики термструктуры для 36+ активов (8 категорий: валюта, индексы, энергоносители, металлы, акции, агро, крипто, облигации). Сравнение с историей | Для анализа контанго/бэквордации |
| **OI Физ/Юр** | Открытый интерес: физики vs юрики. Net позиции, графики истории. Аналог CFTC COT для MOEX | Контрарный индикатор |

### 2.4 Арбитраж & RV

Главная уникальность портала. Две подгруппы:

**Арбитраж (гарантированная конвергенция):**

| Подраздел | Что делает |
|---|---|
| **Cash-and-Carry** | Implied rate фьючерсов vs КС/RUSFAR. Зелёный = зарабатываешь больше депозита |
| **Reverse C&C** | Зеркало: когда фьючерс дешевле спота |
| **Calendar FX** | Спред ближнего vs дальнего валютного фьючерса. Нейтрален к движению валюты |
| **Cross-Instrument** | Big vs mini контракты одного индекса (MX vs MXI, RI vs RH) |
| **Cross-FX** | Треугольник CR/Si/Eu — когда USDRUB ≠ CNYRUB × USDCNY |
| **Cross-Gold** | GL(руб) vs GD(долл) — implied USDRUB из золота vs рынок |
| **CIP Arb** | Covered Interest Parity: implied USDCNY из Si/CR vs спот |

**Relative Value (без гарантии конвергенции):**

| Подраздел | Что делает |
|---|---|
| **Calendar Commodity** | Спреды нефти, газа, металлов между экспирациями |
| **Curve Shape** | Классификация: контанго / бэквордация / смешанная |
| **Rate Spread** | Implied rate vs КС для всех фьючерсов |

### 2.5 Стратегии

| Подраздел | Что делает |
|---|---|
| **Преф/Обычка** | Mean reversion: спред преф vs обычка. Бэктест 12+ лет, Sharpe 1.0-1.3 |
| **Сектор Ротация** | Long top-2 / Short bottom-2 секторов по 3M momentum. Бэктест 15 лет |
| **Momentum** | Cross-asset dual momentum: STRONG BUY / BUY / HOLD / AVOID |
| **Dividend Capture** | Календарь дивидендных отсечек. Tier 1 (100% Win Rate) и Tier 2. ~8%/сделку |

### 2.6 Фонды

| Подраздел | Что делает |
|---|---|
| **Таблица** | 500+ ПИФов: БПИФ, ОПИФ, ИПИФ, ЗПИФ. Доходности 1M/3M/YTD/1Y/3Y, NAV, комиссия УК, приток/отток |
| **Heatmap** | Тепловая карта 100 крупнейших фондов: цвет = доходность, размер = NAV |

### 2.7 Макро

Графики макроэкономических индикаторов РФ (КС ЦБ, инфляция и др.). Dropdown + периоды 1Y/3Y/5Y/10Y/MAX.

### 2.8 Календарь

Единый календарь на 180 дней: 6389 событий (купоны, погашения, решения ЦБ, дивиденды, экспирации фьючерсов). Фильтры по типу, периоду, тикеру.

### 5.3 Рынок

**TAM (Total Addressable Market):**
- 39.4 млн брокерских счетов на MOEX (октябрь 2025)
- 10.2 млн активных инвесторов (хотя бы 1 сделка в год, 2024)
- 3.5 млн активных ежемесячно
- Доля частных инвесторов на FORTS: 62%

**SAM (Serviceable Addressable Market):**
- Целевая аудитория Packman: трейдеры деривативов + облигационные инвесторы
- Оценка: ~500-800 тыс. человек (активные на FORTS + активные облигационные)

**SOM (Serviceable Obtainable Market):**
- При бесплатной модели + нишевой специализации (арбитраж): 5-20 тыс. пользователей в первые 1-2 года
- При платной модели (freemium, 500-1500 ₽/мес): конверсия 2-5% → 100-1000 платящих

**Ценовой бенчмарк:**
- Бесплатные: TradingView (базовый), Smart-Lab, Conomy, Dohod.ru
- 500-1500 ₽/мес: Finrange, FinanceMarker, BlackTerminal
- 5000-15000 ₽/мес: Cbonds, Bloomberg Terminal (для институционалов)

### 5.4 Уникальные преимущества Packman Monitor

1. **Арбитражный фокус** — единственная платформа с real-time cash-and-carry, CIP, cross-FX, calendar spreads для MOEX. Конкурентов в этой нише **нет**
2. **All-in-one** — акции + облигации + фьючерсы + фонды + фундаментал + стратегии в одном месте
3. **Бесплатно** — всё без регистрации и подписки
4. **Real-time** — обновление каждые 3 секунды (конкуренты: TradingView ~15 мин бесплатно)
5. **OI физ/юр** — аналог CFTC COT для MOEX, нет у конкурентов
6. **Стратегии с бэктестами** — преф/обычка (12+ лет), ротация секторов (15 лет)
7. **Календарь** — единый календарь 6389 событий (купоны, погашения, ЦБ, дивиденды)
8. **ОФЗ кривая с историей** — интерактивная кривая доходности за 13 лет

## 6. Структура табов (актуальная, февраль 2026)

