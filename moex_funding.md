# Архитектура модуля фандинга MOEX

## Что такое фандинг и зачем он нужен

Фандинг — механизм удержания цены **вечного фьючерса** рядом со спотовой ценой базового актива. Начисляется/списывается **раз в день** в вечерний клиринг (~19:00 мск). Лонги платят шортам или наоборот — в зависимости от направления отклонения.

---

## Инструменты с фандингом на MOEX

| Тикер | Базовый актив | Клиринг |
|-------|--------------|---------|
| USDRUBF | USD/RUB | 18:50 |
| EURRUBF | EUR/RUB | 18:50 |
| CNYRUBF | CNY/RUB | 18:50 |
| GLDRUBF | Золото | 18:40 |
| IMOEXF | Индекс МосБиржи | 18:40 |
| SBERF, GAZPF и др. | Акции | 18:40 |

---

## Основная формула фандинга

```
Funding = MIN(L2; MAX(-L2; MIN(-L1, D) + MAX(L1, D)))
```

### Переменные

| Переменная | Описание |
|-----------|---------|
| `D` | Отклонение: среднее(Цена ВФ − Цена спота) за каждую минуту с 10:00 до клиринга, исключая промежуточный клиринг |
| `L1` | Допустимое отклонение = K1 × Цена спота. Внутри этого диапазона фандинг = 0 |
| `L2` | Максимальный фандинг = K2 × Цена спота. Фандинг не может превысить это значение |
| `K1, K2` | Статичные параметры, свои для каждого контракта (берутся из спецификации) |

### Логика поведения

| Условие | Результат |
|---------|----------|
| `|D| < L1` | фандинг = 0 |
| `L1 < |D| < L1+L2` | фандинг = D − L1 (по знаку D) |
| `|D| > L1+L2` | фандинг = L2 (максимум, по знаку D) |

---

## Как считается D (отклонение) — пошагово

1. Каждую минуту с 10:00 до вечернего клиринга биржа фиксирует:
   - **Цена ВФ** = медиана из 12 срезов (bid, ask, last) за эту минуту
   - **Цена спота** = последнее значение базового актива
   - Разница за минуту = Цена ВФ − Цена спота

2. D = среднее арифметическое всех минутных разниц за день (исключая время промежуточного клиринга)

```
D = avg(ВФ₁ - Спот₁, ВФ₂ - Спот₂, ..., ВФₙ - Споtₙ)
```

---

## Как фандинг влияет на позицию: кто платит, кто получает

| Ситуация | D | Фандинг | Лонг | Шорт |
|----------|---|---------|------|------|
| ВФ дороже спота | D > L1 | Положительный | **Платит** | **Получает** |
| ВФ дешевле спота | D < −L1 | Отрицательный | **Получает** | **Платит** |
| ВФ ≈ спот | \|D\| < L1 | 0 | Ничего | Ничего |

**Смысл:** если все хотят купить фьючерс (он дороже спота) — лонги платят штраф шортам. Это давит цену вниз, к споту.

---

## Торговая логика: покупать или шортить на основе фандинга

### Высокий положительный фандинг (ВФ сильно дороже спота)
- Лонги платят шортам каждый день
- **Выгодно держать шорт** — получаешь фандинг пассивно
- Рынок перегрет вверх → возможна коррекция
- Стратегия: **шорт ВФ + лонг спота** (cash-and-carry наоборот)

### Высокий отрицательный фандинг (ВФ сильно дешевле спота)
- Шорты платят лонгам каждый день
- **Выгодно держать лонг** — получаешь фандинг пассивно
- Рынок перепродан → возможен отскок
- Стратегия: **лонг ВФ + шорт спота**

### Нулевой или малый фандинг (\|D\| < L1)
- Нет смысла торговать именно из-за фандинга
- Фандинг не влияет на P&L

## Пример расчёта USDRUBF

Параметры: K1 = 0.1%, K2 = 0.15%, Цена спота = 87 руб.

```
L1 = 0.001 × 87 = 0.087 руб.
L2 = 0.0015 × 87 = 0.1305 руб.
```

| D (отклонение) | Фандинг | Комментарий |
|---------------|---------|------------|
| +0.05 руб. | 0 | Меньше L1, фандинга нет |
| +0.15 руб. | +0.063 руб. | D − L1 = 0.15 − 0.087 |
| +0.25 руб. | +0.1305 руб. | Достигнут максимум L2 |
| −0.10 руб. | −0.013 руб. | D + L1 = −0.10 + 0.087 |

---

## Цена вечного фьючерса (ВФ) в формуле D

Каждую минуту берётся **медиана** из 12 срезов bid/ask/last.  
Цена базового актива — последнее значение спота.

---

## Итоговый фандинг для позиции

```
Фандинг позиции = Размер позиции (контрактов) × Фандинг × Лот
```

**Пример:** CNYRUBF, фандинг = 0.0015 руб., позиция = −2 контракта, лот = 1000  
→ Фандинг = 0.0015 × 1000 × 2 = **3 руб. к получению**  
(позиция короткая, фандинг положительный → шорты получают)

---

## Формула вариационной маржи в вечерний клиринг

```
ВМ = Переоценка позиции − Фандинг × Лот [+ Дивидендная поправка × Лот]
```

> Дивидендная поправка — только для IMOEXF (индекс дивидендов IMOEXDIV)

### Промежуточный клиринг (без фандинга)

```
ВМ = Переоценка позиции
```

---

## Индикативный фандинг (real-time)

- Считается **каждую минуту** по той же формуле
- Публикуется в поле `swap_rate` в потоке `FORTS_COMMON_REPL` (шлюз биржи)
- Отображается на странице контракта на moex.com
- Нужен для мониторинга в течение дня до клиринга

---

## Откуда брать цену спота

### По типу инструмента

| Инструмент | Источник спота | Резерв |
|-----------|---------------|--------|
| USDRUBF, EURRUBF, CNYRUBF | Валютный рынок MOEX (секция CURR), расчётная цена предыдущего ВК | Курс ЦБ на следующий день |
| SBERF, GAZPF и др. (акции) | Цена закрытия акции на фондовом рынке MOEX (секция TQBR) | — |
| IMOEXF (индекс) | Последнее значение индекса IMOEX | — |
| GLDRUBF (золото) | Цены на золото с валютного рынка MOEX | — |

### API endpoints (бесплатный публичный MOEX ISS, авторизация не нужна)

**Валюта — курс USD/RUB TOM:**
```
GET https://iss.moex.com/iss/engines/currency/markets/selt/securities/USD000UTSTOM.json
```

**Акция — цена Сбера:**
```
GET https://iss.moex.com/iss/engines/stock/markets/shares/securities/SBER.json
```

**Индекс IMOEX:**
```
GET https://iss.moex.com/iss/engines/stock/markets/index/securities/IMOEX.json
```

**Минутные свечи для расчёта D (пример — Сбер):**
```
GET https://iss.moex.com/iss/engines/stock/markets/shares/securities/SBER/candles.json?interval=1
```

**Минутные свечи вечного фьючерса (пример — USDRUBF):**
```
GET https://iss.moex.com/iss/engines/futures/markets/forts/securities/USDRUBF/candles.json?interval=1
```

**История фандинга (swap_rate) из потока шлюза:**
```
FORTS_COMMON_REPL → поле swap_rate
```
> Для получения swap_rate через шлюз нужно подключение к FAST-протоколу MOEX. Для простых задач достаточно страницы итогов торгов на moex.com.

## Источники данных для кода

| Данные | Где брать |
|--------|----------|
| Текущий индикативный фандинг | `FORTS_COMMON_REPL.swap_rate` (шлюз) |
| История фандинга | Страница итогов торгов контракта на moex.com |
| K1, K2, лот | Спецификация контракта (moex.com → Срочный рынок → Инструменты) |
| Цена спота | Валютный/фондовый рынок MOEX |
| Цена ВФ | Срочный рынок MOEX |
