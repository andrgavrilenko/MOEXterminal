# Codex_Учебник_Как_Работать_С_Порталом.md

## Важно перед стартом

Это учебный материал, не инвестиционная рекомендация.

Принятые допущения для этого документа:
- Комиссии и проскальзывание **игнорируем**.
- Фокус на логике сигналов и конструкции позиций long/short.

---

## 1. Единый формат работы по любому дашборду

1. Проверяешь сигнал (формула/таблица/колонка Profit).
2. Определяешь конструкцию (какие ноги long/short).
3. Открываешь ноги в указанном порядке.
4. Сопровождаешь до условия выхода.

---

## 2. Карта дашбордов: что смотреть, как входить, что делать

| Дашборд | Что это | Что смотреть | Условие входа | Позиция (long/short) | Порядок исполнения | Выход |
|---|---|---|---|---|---|---|
| Арбитраж / Cash-and-Carry | Конвергенция спот-фьюч | `Implied Rate`, `Премия к КС`, ликвидность | `Премия к КС > 0` | `Long spot + Short futures` | 1) Long spot 2) Short futures | Экспирация или сжатие премии к цели |
| Арбитраж / Reverse C&C | Обратная конвергенция | `Implied Rate`, `Премия к КС` | `Implied << КС` (бэквордация) | `Short spot + Long futures` | 1) Short spot 2) Long futures | Экспирация или возврат implied к норме |
| Арбитраж / Calendar FX | Спред двух экспираций одного актива | `Δ Rate`, `ABS spread`, направление | Большой `Δ Rate` (обычно топ таблицы) | Если дальний дорогой: `Long near + Short far`; иначе наоборот | 1) Открыть недооцененный контракт 2) Открыть переоцененный | Когда `Δ Rate` сужается до цели |
| Арбитраж / Cross-Instrument | Пары big/mini одного базового | `Spread`, `Rate Diff`, `Profit` | Значимое отклонение spread | `Short переоцененный / Long недооцененный` | 1) Short дорогой контракт 2) Long дешевый | При возврате spread к среднему |
| Арбитраж / Cross-FX | Треугольный FX-арбитраж (Si/CR/Eu) | `Max Δ`, арб-пары, действие | Высокий `Max Δ` и явное направление в таблице | По сигналу строки (например `Long Si + Short CR`, либо обратная) | 1) Открыть фьючерсную пару 2) Добавить хедж-ногу при необходимости | Когда `Max Δ` нормализуется |
| Арбитраж / Cross-Gold | GL vs GD через USD компонент | `Impl.USDRUB` vs `Mkt.USDRUB`, `Расхождение` | Значимое расхождение impl и market | Если GL переоценен: `Short GL + Long GD + Long Si`; если недооценен — наоборот | 1) Открыть GL/GD 2) Добавить валютный хедж Si | При закрытии расхождения |
| Арбитраж / CIP Arb | Covered Interest Parity (Si/CR) | `Fwd Premium`, отклонение от среднего, `Profit` | Сильное отклонение `Fwd Premium` от своей нормы | Направление по таблице CIP (где премия аномальна) | 1) Открыть Si/CR 2) Добавить валютную компенсирующую ногу при необходимости | Когда `Fwd Premium` возвращается к среднему |
| Relative Value / Calendar Commodity | Спреды по сырью между экспирациями | `Δ Rate`, `ABS spread` | Верхние отклонения ранжирования | `Long near/Short far` или наоборот по знаку `Δ Rate` | 1) Открыть недооцененную ногу 2) Переоцененную | При нормализации спреда |
| Relative Value / Curve Shape | Форма кривой срочности | `Контанго/Бэквордация/Смешанная`, величина спреда | Изменение формы + экстремум спреда | Выбор пар внутри кривой в сторону ожидаемой нормализации | 1) Выбрать ближний/дальний 2) Открыть календарный спред | При возврате формы/спреда |
| Relative Value / Rate Spread | Рейтинг carry-идей | `Profit`, `Δ rate`, действие | Топ ранжирования по profit | Следовать указанной паре long/short | 1) Проверить актив 2) Исполнить пару по таблице | По целевому сжатию spread |
| Div Arb & Carry | Дивидендные/carry идеи по акциям | `prog_div`, `div_yield`, front/back rates, отсечка | Высокий `div_yield` + выраженный carry-дисбаланс | Базово: `Short stock + Long stock futures` (или обратная по сигналу) | 1) Открыть акцию 2) Открыть фьючерс | После див-события и/или нормализации basis |
| Market Data / Акции | Поток котировок и фильтры по акциям | Оборот, сделки, спред, сектор | Не вход, а фильтр ликвидности для идей | Направление не задает | 1) Найти ликвидный инструмент 2) Вернуться в сигнальный дашборд | Используется как pre-trade фильтр |
| Market Data / Корп.бонды | Доходность/дюрация корп.бумаг | YTM, duration, g-spread | Идеи от экстремумов доходность/дюрация | Обычно RV-пары по кривой | 1) Выбор бумаг по дюрации 2) Построение спред-пары | При сужении доходностного спреда |
| Market Data / ОФЗ | Кривая госдолга | Доходность, дюрация, выпуски | Экстремумы кривой ОФЗ | RV-позиции по участкам кривой | 1) Выбор участка 2) Пара/корзина | При возврате участка к среднему |
| Market Data / Фьючерсы | Сводка фьючерсной ликвидности | OI, volume, disbalance, spread | Не вход, а фильтр исполнимости идей | Направление не задает | 1) Проверка стакана/объема 2) Подтверждение входа | Используется как pre-trade фильтр |
| Консенсус Акции | Оценка upside и мультипликаторов | `Upside`, `Reco`, `P/E`, `EV/EBITDA` | Высокий upside + поддержка по качеству | Long идеи в сильных бумагах, short — в слабых | 1) Отбор кандидатов 2) Подтверждение в Market Data/Стратегиях | При достижении target или пересмотре консенсуса |
| Стратегии / Преф-Обычка | Mean reversion в парах pref/common | `z-score`, `ratio`, `signal` | Z-экстремумы пары | Short дорогая ножка / Long дешевая | 1) Открыть дорогую в short 2) Дешевую в long | При возврате z-score к нулю/среднему |
| Стратегии / Сектор Ротация | Momentum L/S по секторам | 1M/3M/6M/12M, `Signal` | Явная дивергенция лидеров/аутсайдеров | `Long лидеры / Short отстающие` | 1) Long basket 2) Short basket | При смене ранга и затухании импульса |
| Стратегии / Momentum | Cross-asset momentum board | `Signal`, `Reason`, перф-окна | STRONG BUY/BUY/AVOID по правилам модели | Long сильные, short слабые | 1) Открыть по сигналу 2) Ребаланс по обновлению | При смене сигнала |
| Стратегии / Дивиденды | Календарные див-сигналы | Tier, Z, Avg%, WinRate, статус | Бумаги верхних tier + валидный статус | Чаще event-driven позиции вокруг отсечки | 1) Вход до окна события 2) Выход по плану события | По завершению див-окна |
| Макро | Макро-режим рынка | КС ЦБ, инфляц./ставочные тренды | Не прямой вход, а risk-regime фильтр | Направление не задает напрямую | 1) Определить режим 2) Подстроить горизонт/плечо | При смене макро-режима |
| Объёмы | Аномалии торговой активности | RVOL, x1d/x5d/x10d | Резкий всплеск объема в инструменте идеи | Усиливает или ослабляет основной сигнал | 1) Проверить аномалию 2) Подтвердить сделку | Как фильтр подтверждения/отмены |
| OI Физ/Юр | Позиционирование участников | `физ long/short/net`, `юр long/short/net` | Экстремальный перекос позиций | Контртренд/тренд по твоему правилу | 1) Определить перекос 2) Привязать к сигналу из Арбитраж/RV | При нормализации OI-перекоса |
| Кривые | Term structure по классам активов | Форма кривой сегодня vs сдвиги (`-1D/-1W/-1M/-1Y`) | Деформация формы относительно истории | Календарные спреды в сторону нормализации | 1) Выбрать класс/актив 2) Открыть near/far пару | При возврате кривой к исторической форме |

## 3. Мини-playbook на день

1. `Арбитраж/Cash-and-Carry`: выбрать топ-идеи по премии.
2. `Market Data/Фьючерсы`: проверить исполнимость ног.
3. `Calendar FX` и `Кривые`: проверить, не конфликтует ли структура сроков.
4. `CIP/Cross-FX/Cross-Gold`: добавить только если нужна multi-leg идея.
5. `Макро` + `OI` + `Объёмы`: подтвердить режим и не входить против сильного фонового сигнала.
6. Зафиксировать план: вход, условие выхода, критический уровень отмены идеи.

## 4. Приоритет по сложности (практический)

1. Cash-and-Carry
2. Cross-Instrument (big/mini)
3. Calendar FX
4. Reverse C&C
5. Div Arb & Carry
6. Cross-Gold
7. CIP / Cross-FX multi-leg

Начинать лучше с 1-3, затем переходить к 4-7.
